# Nova Orchestrate

스킬 `orchestrator`를 실행한다.

사용자 입력을 그대로 스킬에 전달한다.

예시:
- /nova:orchestrate "proptech-lab에 건폐율 시각화 추가"
- /nova:orchestrate "nova-landing 한국어 버전 추가" --design-only
- /nova:orchestrate "3개 프로젝트에 다크모드 통일" --strict
